The REST API exposes market data and account actions for trading clients.

<APIEndpoint method="GET" path="/v1/markets" description="List supported perpetual markets" />
<APIEndpoint method="GET" path="/v1/orderbook/:market" description="Read top-of-book depth" />
<APIEndpoint method="POST" path="/v1/orders" description="Place a signed order intent" />
<APIEndpoint method="DELETE" path="/v1/orders/:id" description="Cancel an open order" />
<APIEndpoint method="GET" path="/v1/account" description="Read balances and positions" />

## Authentication

Private endpoints require a wallet-derived session signature. Public market data
endpoints can be requested without authentication.
